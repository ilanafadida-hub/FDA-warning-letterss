🔬 FDA Warning Letter Analysis System

Search, filter, visualize, and analyze FDA warning letters — all in one dashboard

🚀 Getting Started

1

Open the Dashboard

Click the shared link to open the dashboard in your browser. No installation needed.

2

Add API Key (Optional)

In the sidebar, paste your OpenAI API key to unlock AI-powered Q&A. Without a key, everything still works using rule-based analysis.

3

Explore & Analyze

Use the tabs to search letters, view trends, get insights, and read full letter details.

🔑 Getting an OpenAI key: Go to platform.openai.com/api-keys → Create new secret key → Copy it → Paste in the sidebar. Your key stays in your browser session only and is never stored.

❓ FAQ

Do I need an API key?

No. The dashboard is fully functional without one. The API key only adds AI-powered Q&A answers.

Is my API key safe?

Yes. It stays in your browser session only and is never saved to disk or sent to our servers.

How much does it cost?

Each AI Q&A question costs roughly $0.001–$0.01 using GPT-4o-mini. Very affordable for normal use.

How often is data updated?

Use the Refresh button in the sidebar to fetch new letters from FDA. Data is updated incrementally.
